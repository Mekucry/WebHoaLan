# Cài đặt 

- Bước 1: `git clone https://github.com/Mekucry/WebHoaLan.git`
- Bước 2: Import các file sql
- Bước 3: `cd NhanDangHoa`
- Bước 4: 'python NhanDangHoa.py'
- Bước 5: Tạo terminal mới rồi `cd Web`
- Bước 6: `node servers.js`
- Bước 7:  Sau khi thông báo kết nối thành công thì truy cập vào 'http://localhost:3000'
